# VIBE GAME 教案・第 4 集「美術整合」— 分鏡腳本

- **影片型態**：純螢幕＋旁白配音（SVG 動畫＋簡報圖片素材＋螢幕錄影）
- **製作工具**：Remotion（SVG 動畫／圖解／佔位卡）＋ 螢幕錄影（Unity／Figma／AI 生圖流程）
- **預估片長**：固定旁白約 3～4 分鐘＋四段實際操作錄影＋結尾，總長依操作內容調整
- **視覺基調**：白底／米白底、深灰文字、黃色重點色；延續前 3 集的留白、卡片與流程圖風格
- **呈現方式圖例**：`SVG動畫` / `圖片素材` / `螢幕錄影` / `文字卡` / `佔位卡`

> 逐字稿中「AI」「Unity」「Figma」「Sprite Sheet」「PPU」等詞已統一大小寫；明顯口誤如 `1unty` 修正為 `1 unit`。

Unity 有一個叫做 PPU 的欄位，只要設定這個欄位，就可以讓 unit 和 px 之間的轉換是正常的。
