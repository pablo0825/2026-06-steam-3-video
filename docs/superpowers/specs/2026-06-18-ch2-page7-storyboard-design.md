# Chapter 2 Page 7 Storyboard Design

## Goal

實作第二章第 7 頁 S18–S22，將「分鏡圖的用途、形式、團隊共識與繪製重點」串成一段連續的 49 秒教學動畫。

## Scope

- Composition ID：`Ch2-Page7-Storyboard`
- 畫面尺寸：1920 × 1080
- 幀率：30 fps
- 總長：1470 frames（49 秒）
- 場景範圍：S18–S22
- 延續第二章既有的白底、深灰文字、暖灰輔助線與黃色重點色
- 所有動畫由 `useCurrentFrame()`、`interpolate()`、`spring()` 控制，不使用 CSS transition 或 CSS animation

## Narrative Structure

採用單一連續場景，不將五個鏡頭做成彼此獨立的投影片：

1. S18 先建立「分鏡圖可以演出玩法與關卡」的基本概念。
2. S19 將示意分鏡轉換成三張真實案例，說明形式多元。
3. S20 從素材牆聚焦兩張差異最大的案例，讓觀眾看清楚細節。
4. S21 將分鏡卡轉化成團隊溝通媒介，說明凝聚共識的功能。
5. S22 收束為四項可執行的繪製重點。

每段轉場沿用上一段的視覺元素，避免整頁清空後重新進場。

## S18 — Storyboard Introduction

### Copy

- 標題：`Storyboard 分鏡圖`
- 定義句：`演出玩法・關卡設計`

### Layout

- 標題位於畫面上方中央，沿用第二章標題字級與字重。
- `Storyboard` 使用黃色，`分鏡圖` 使用深灰色。
- 定義句位於標題下方，以較小的深灰字呈現。
- 下方放置三個等寬分鏡格，使用暖灰邊框。
- 三格內容依序表示：
  1. 玩家位於平台起點。
  2. 玩家依箭頭跳躍或移動。
  3. 玩家抵達終點旗幟。

### Animation

1. 標題先以輕微 spring 上移進場。
2. 定義句延後淡入。
3. 三格外框由左至右依序畫出。
4. 每格中的角色、平台與箭頭在該格外框完成後出現。
5. S19 開始前，三格縮小並靠向畫面中央，作為真實素材牆的轉場起點。

## S19 — Multiple Formats

### Copy

- 標題：`形式多元・無固定格式`
- 章印：`看得懂最重要`

### Assets

- `public/02-遊戲設計/storyboard-sample-1.png`
- `public/02-遊戲設計/storyboard-sample-2.png`
- `public/02-遊戲設計/storyboard-sample-3.png`

### Layout

- 三張圖片以素材牆形式並排，但刻意保留不同尺寸與輕微旋轉角度。
- `sample-1` 放左側，呈現黑白格狀草圖。
- `sample-2` 放中央，呈現乾淨的數位流程圖。
- `sample-3` 放右側，呈現彩色手繪草稿。
- 圖片使用白色卡片底、細邊框與低強度陰影，避免與白色背景混在一起。
- 黃色章印位於素材牆右下方，不遮住主要內容。

### Animation

1. S18 的三格示意淡出。
2. 三張真實圖片由左至右依序淡入並落到各自角度。
3. 標題在第一張圖片前完成進場。
4. 三張圖片都穩定後，章印以較有重量感的 spring 縮放落下。

## S20 — Detail Focus

### Selected Samples

- 第一張：`storyboard-sample-1.png`
- 第二張：`storyboard-sample-3.png`

兩張分別代表簡單黑白分鏡與彩色潦草手繪，差異最大，能延續 S19「形式多元」的訊息。

### Layout and Motion

- 素材牆先降低亮度與透明度，當前圖片移至中央並放大。
- 第一階段聚焦 `sample-1`：
  - 以圖片左上至中間區域為主要裁切內容。
  - 保留數個連續分鏡格，讓觀眾看出角色移動與關卡設計。
- 第二階段聚焦 `sample-3`：
  - 以第一排三格為主要裁切內容。
  - 保留紅藍筆跡、箭頭與手寫註記。
- 每張圖片使用約 1.03 至 1.08 的緩慢 Ken Burns 縮放。
- 兩張圖片之間使用短暫交叉淡化，不使用滑動輪播。
- 不新增說明卡或大量文字，讓圖片成為唯一焦點。

## S21 — Team Consensus

### Copy

- 核心詞：`凝聚團隊共識`
- 對話框：`我們想的一樣嗎？`
- 確認文字：`想像一致`

### Layout

- S20 最後一張圖片縮小成中央的分鏡卡。
- 三個抽象成員頭像分布於左、右與下方，使用簡單 SVG 頭像，不設定具體性別或角色。
- 頭像以深灰、暖灰與低飽和藍區分，黃色僅保留給確認狀態。
- 分鏡卡與頭像之間以細線連接。
- 左右各出現一個對話框，內容相同：`我們想的一樣嗎？`
- 最終在分鏡卡下方顯示黃色勾號、`想像一致` 與主標 `凝聚團隊共識`。

### Animation

1. 分鏡卡從 S20 的放大圖片縮回中央。
2. 三個頭像依序向中央聚攏。
3. 連接線隨頭像靠近逐段畫出。
4. 兩個對話框先後彈出。
5. 對話框短暫停留後，黃色勾號畫出。
6. 勾號完成後才顯示 `想像一致` 與 `凝聚團隊共識`。

重點順序必須是「看分鏡 → 詢問 → 確認」，不能一開始就顯示成功狀態。

## S22 — Storyboard Checklist

### Copy

- 標題：`繪製分鏡圖的重點`
- 四項重點：
  1. `完整開始與結尾`
  2. `情境連貫`
  3. `去複雜化`
  4. `夥伴看得懂`

### Layout

- S21 的中央分鏡卡縮小成標題旁的分鏡圖示，作為內容延續。
- 四個重點使用 2 × 2 卡片排列，避免單排文字過長。
- 每張卡片包含一個 SVG 小圖示與一行文字：
  1. 起點與終點旗幟。
  2. 三個連續畫格與方向箭頭。
  3. 方塊、圓形與簡化線條。
  4. 兩個人物與確認勾號。
- 卡片預設使用暖灰邊框，進場瞬間以黃色邊框提示，穩定後回到暖灰。
- 最後一張「夥伴看得懂」保留黃色圖示或文字，作為整頁結論。

### Animation

1. 標題先進場。
2. 四張卡片依閱讀順序左上、右上、左下、右下逐一彈入。
3. 每張卡片中的 SVG 圖示比文字早 4–6 frames 出現。
4. 前三張卡片完成後恢復暖灰，最後一張保留黃色焦點。

### Acceptance Criteria

- S18 必須是標題先進場，再開始繪製三格分鏡。
- S19 的三張素材必須保留明顯不同的原始風格。
- S20 必須只聚焦 `sample-1` 與 `sample-3`，且裁切後仍能辨識分鏡內容。
- S21 必須依序表達「觀看、詢問、確認、形成共識」。
- S22 四項重點必須逐一出現，最後焦點落在「夥伴看得懂」。
- Page 7 與 Page 5、Page 6 的字體、背景、色彩與動畫節奏一致。
